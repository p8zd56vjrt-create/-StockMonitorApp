# 股票盯盘应用

这是一个Android股票盯盘应用，可以实时监控股票价格，并在达到设定的买入或卖出阈值时发送通知。

## 功能特性

- 添加股票代码进行监控
- 实时显示股票价格和涨跌情况
- 设置买入和卖出阈值
- 当股票价格达到阈值时发送通知提醒

## 技术栈

- Java
- Android SDK
- Retrofit（网络请求）
- MPAndroidChart（图表库）

## 构建和运行

1. 使用Android Studio打开项目
2. 同步Gradle依赖
3. 运行应用到Android设备或模拟器

## 使用说明

1. 在应用主界面输入股票代码，点击"添加股票"按钮
2. 股票会显示在列表中，实时更新价格
3. 可以为每个股票设置买入和卖出阈值
4. 当股票价格达到阈值时，应用会发送通知提醒

## 注意事项

- 本应用使用模拟数据进行演示，实际使用时需要替换为真实的股票API
- 阈值设置功能需要根据实际需求进行调整
- 通知功能需要在系统设置中允许应用发送通知
